# wallpaper-bot
wallpaper engine in telegram
https://t.me/livewallpaper_bot

## Features
- Rating system
- Gif and photo support
- Media is not stored on the server 
- Upload your wallpapers
- Wallpapers carousel
- Notifications system
- Profile system
